# HYRAX
A database of UK mountains with modern web and desktop front ends built with .Net 6 as a learning exercise

## The database
The database is taken from the following web site http://www.hills-database.co.uk/ All credit must go to this sites owners who have done a fantastic job of creating and maintaining this database called the Database of British and Irish Hills (DoBIH).

The original Access (tm) database will be deconstructed and key data imported into a SQL Server database ultimately running on Azure. An API will be created to allow access to the data in a structured way.
